# Badge  
> [Badge] component that can be used inside product. Has `OpacityVisible` state, `badgeZero` state.  

## Props  
  
|                       | required | types                 | default   |
| --------------------- | -------- | --------------------- | --------- |
| count                 |          | number                |           |
| color                 |          | string                |           |
| maximumValue          |          | number                |           |
| showZero              |          | boolean               |           |
| opacityVisible        |          | boolean               |           |

  
## Installation  
  
```sh
yarn add dooboo-ui
```
  
## Getting started  
  
- Import

  ```javascript
  import { Badge } from 'dooboo-ui';
  ```
